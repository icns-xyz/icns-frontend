# icns-frontend

## Development

Add an .env file and fill the following variables or manually add them in your terminal.

```
TWITTER_CLIENT_ID=...
TWITTER_CLIENT_SECRET=...
TWITTER_AUTH_STATE=...
TWITTER_AUTH_CALLBACK_URI=...

IRON_PASSWORD=...

ICNS_VERIFIER_URI=...
```

Run the development server:

```bash
yarn
yarn dev
```
